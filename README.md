# Attorney & Legal Consultant Website Template

## Overview
This project is a premium, fully responsive, and highly dynamic web template designed specifically for law firms, solo attorneys, and legal consultants (currently branded for Attorney Y Sai Yashwanth Reddy). It provides a professional online presence with a comprehensive suite of features to manage both public-facing marketing and secure client interactions.

## Key Features

### 1. Public Facing Pages
- **Home (2 variations)**: High-impact landing pages with hero animations, particle effects, and dynamic counting statistics.
- **About & Practice Areas**: Detailed pages highlighting attorney credentials and specialized legal services (Family Law, Estate Planning, etc.).
- **Blog / Insights**: A searchable, filterable repository of legal articles and insights.
- **Contact**: A fully functional contact form with embedded maps and interactive practice area chips.

### 2. Secure Portals
- **Client Portal**: A secure dashboard for clients to view their case status, upload documents, send encrypted messages, and pay retainers/invoices.
- **Admin Portal**: A backend interface for the law firm to manage appointments, client documents, secure messages, and billing.

### 3. Advanced UI/UX
- **Theming**: Built-in support for multiple themes (Navy Dark Mode, Ivory Light Mode, and a secondary Light Mode).
- **RTL Support**: Full Right-to-Left (RTL) language support with a toggle button that mirrors the entire UI, including sidebars and mobile menus.
- **Micro-interactions**: Smooth page transitions, hover effects, floating particles, and scroll-triggered animations.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices with a custom off-canvas mobile menu drawer.

## Technologies Used
- **HTML5**: Semantic and accessible markup.
- **CSS3**: Vanilla CSS utilizing modern features like CSS Variables (Custom Properties), Flexbox, Grid, and Keyframe Animations.
- **JavaScript (Vanilla)**: DOM manipulation, Intersection Observers for scroll animations, and dynamic CSS injection for complex mobile animations.
- **FontAwesome**: Scalable vector icons for the UI.
- **Google Fonts**: Premium typography (Playfair Display, Cormorant Garamond, Inter).

## Getting Started
To view or edit this project:
1. Clone or download the repository.
2. Open any HTML file inside the `pages/` directory (e.g., `pages/index.html`) in a modern web browser.
3. No build tools or Node.js required—this is a pure Vanilla HTML/CSS/JS template.
